Backstory

A fun side project that generates boilerplate web application code.

  • It generate code that is useful to me
  • Keeps me busy
  • Lets me tinker with different tools (like htmx)
  • I can approach problems in new (to me) ways

I have iterated on this idea many times over several years (taking multi month breaks in between). The current version you are seeing is the latest challenge for myself - a fully SSR site with state exclusive to the server. The only client side JS is htmx.
